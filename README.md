###### (All the features have been implemented)
> Web Application
  - Live map data
  - Multi lingual support
  - Intutive UI
  - Notification Management
  - Data vizualisaton
  - Admin model for easy control
  - Forcast model
  - Realtime IOT data
  - Data Encryption
  - Progressive web application support
  - Responsive Design
  
> Mobile Application
  - Intutive UI
  - Notification Management
  - Nearby API
  - Push Notification
  - Realtime IOT data

> IOT Device
  - Cost Effective Design
  - Siren Alert
  - Lossless Data Streaming
  - Firewall and Encryption to prevent unauthorized access

  - Realtime Dashboard
  - Scales with V-Sat connection
 
> Notification 
  - Localized Call Alert
  - Localized SMS Alert
  - Mail Alert
  - Twitter Bot
  - Telegram Bot
